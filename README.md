# HEART Metrics

AI that helps managers retain top talent. HEART Metrics turns everyday employee signals into early burnout warnings so managers can intervene before good people leave.

**Live site:** [https://heartmetrics.us](https://heartmetrics.us)
**Live demo:** [https://heartmetrics.us/demo](https://heartmetrics.us/demo)

## Repository structure

- `index.html` — marketing homepage served at `heartmetrics.us`
- `demo/index.html` — interactive prototype served at `heartmetrics.us/demo`
- `favicon.svg` — brand mark (shared by both pages)
- `CNAME` — GitHub Pages custom domain config

## About

HEART Metrics combines four dimensions of work wellbeing — Workload, Recognition, Feedback, and Growth — into a single Work Wellbeing Index (WWI) per employee. Grounded in the U.S. Surgeon General's Framework for Workplace Mental Health & Well-Being.

## Stack

Static HTML. React 18 + Tailwind CSS, loaded from CDN. No build step required.

All employee data in the live demo is synthetic and for illustration purposes only.

## Run locally

```bash
git clone https://github.com/vipulrfid/heartmetrics.git
cd heartmetrics
# Open index.html (homepage) or demo/index.html (prototype) in any browser
```
